FILE USAGE GUIDE

FOR MANUSCRIPT SUBMISSION:

  → Use PNG versions for structural figures (2A, 2B)

FOR PRESENTATIONS:
  → Use PNG versions (universal compatibility)
  → SVG can be used in modern presentation software

FOR EDITING/MODIFICATION:
  → Use SVG versions in Inkscape (free)
  → Use PDF versions in Adobe Illustrator (commercial)
  → PNG structural figures require PyMOL for re-rendering

FOR PRINT PROOFS:
  → Use PDF versions (highest fidelity)
  → Print at 180 mm width for double-column layout

FOR WEB/SUPPLEMENTARY:
  → Use PNG versions (smaller file sizes)
  → Consider creating 150 DPI versions for web display

================================================================================
EDITING RECOMMENDATIONS
================================================================================

To modify vector figures (1, 3, 4, 5):
  1. Open SVG file in Inkscape or PDF in Illustrator
  2. Edit text, colors, or layout as needed
  3. Re-export as PDF for submission

To modify structural figures (2A, 2B):
  1. PyMOL session: /storage/kiran-stuff/aaRS/structural_figures/v2/structural_analysis.pse
  2. Adjust view, colors, or rendering in PyMOL
  3. Re-render at 2000×2000, 300 DPI (or higher if needed)

To convert formats:
  PNG → TIFF:  convert input.png -compress lzw output.tiff
  SVG → PDF:   inkscape input.svg --export-pdf=output.pdf
  PDF → EPS:   pdftops -eps input.pdf output.eps
